How It Works

From idea to code in seconds

Describe what you want. Cipher plans, writes, executes, and iterates — all from your terminal.

STEP 01

You Describe

Type what you want in natural language. No special syntax needed.

STEP 02

Cipher Plans

AI analyzes your request, reads existing code, and creates a strategy.

STEP 03

Writes Code

Complete files written to disk. No placeholders. No incomplete blocks.

STEP 04

Runs Commands

Installs deps, runs tests, starts servers. Full terminal control.

STEP 05

Done

Your code is ready. Review, iterate, or deploy. You're in control.

Features

Everything you need
to build faster

A complete coding agent that understands your project and works alongside you.

Writes Complete Code

Generates full, production-ready files. No placeholders, no "insert code here" shortcuts. Every file is complete.

Runs Terminal Commands

Installs dependencies, runs tests, builds projects, starts servers. Full terminal automation with your approval.

Reads & Edits Files

Understands existing codebases, then writes the complete updated version. Every edit is intentional and complete.

Thinks Before Acting

Plans strategy before writing a single line. See the reasoning, approve the approach, then watch it execute.

Sandboxed Execution

Only touches files in your project directory. Cannot escape your workspace. Your system stays safe.

Setup in minutes

Get Cipher running with your preferred AI provider. Step-by-step guides below.

Installation
1# Windows (PowerShell)
2> irm https://raw.githubusercontent.com
3 /elevenpercent/cipher/master
4 /install.ps1 | iex
5
6# macOS / Linux
7$ curl -fsSL https://cipher.elevenpct.com
8 /install.sh | bash
cipher.config.yaml
1provider: ollama
2model: codellama:7b
3max_tokens: 4096
4
5# Or for OpenAI:
6# provider: openai
7# model: gpt-4o
8# api_key: $OPENAI_API_KEY

Requirements

  • Python 3.10 or higher
  • pip package manager
  • Git (for project integration)
  • Internet connection (for cloud providers)

Usage

Navigate to your project directory and run Cipher with your task:

Terminal
1$ cd my-project
2$ cipher "Create a REST API"
3# Cipher plans, writes, and runs
4# your API automatically

Switching Providers

Change your provider anytime by editing the config or passing a flag:

Terminal
1$ cipher --provider openai
2 --model gpt-4o
3$ cipher --provider anthropic
4 --model claude-3-5-sonnet
