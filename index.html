Quick Start

cd my-project
cip --setup
cip

First Run

On first run, Cipher guides you through selecting an AI provider and model. Selections are saved to ~/.cipher/config.json.

Config File

{
  "provider": "ollama",
  "model": "ollama/qwen3:14b",
  "show_plan": true,
  "show_code": true,
  "show_diff": true,
  "compact_mode": false
}

Environment Variables

VariableDescription
CIPHER_PROVIDERProvider ID (ollama, openai, etc.)
CIPHER_MODELModel ID
OPENAI_API_KEYOpenAI API key
ANTHROPIC_API_KEYAnthropic API key

Commands

cip --setup                    # Interactive setup
cip --provider ollama          # Specify provider
cip --model ollama/qwen3:14b   # Specify model
cip --dir /path/to/project     # Working directory
cip --api-key your-key         # Custom API key

System Prompt

Cipher uses XML-style action tags: <plan>, <write>, <run>, <read>, <ls>. Customizable in cipher/app.py.

Project Isolation

All file paths are validated against the working directory. Attempts to access files outside the project root are rejected.

Common Issues

Command not found: cip

Add Python Scripts to PATH. On Windows: %LOCALAPPDATA%\Programs\Python\Python312\Scripts

Use cip not cipher

Windows has cipher.exe (file encryption) which shadows the command. Use cip instead.

Ollama connection refused

Start Ollama: ollama serve
