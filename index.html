Install

Requires Python 3.10+.

Windows

irm https://cipher.elevenpct.com/install.ps1 | iex

macOS / Linux

curl -fsSL https://cipher.elevenpct.com/install.sh | bash
Tip: Make sure Python is installed first. macOS: brew install python3. Linux: sudo apt install python3 python3-pip.

Quick Start

cd my-project
cip --setup
cip

First Run

Cipher guides you through selecting an AI provider and model. Selections are saved to ~/.cipher/config.json and persist across sessions.

Config File

{
  "provider": "deepseek",
  "model": "deepseek/deepseek-chat",
  "show_plan": true,
  "show_code": true,
  "show_diff": true,
  "compact_mode": false
}

Edit directly or use Ctrl+S in the TUI.

Environment Variables

VariableDescription
CIPHER_PROVIDERProvider ID
CIPHER_MODELModel ID
OPENAI_API_KEYOpenAI key
ANTHROPIC_API_KEYAnthropic key

Commands

cip --setup                    # Interactive setup
cip --provider deepseek          # Specify provider
cip --model deepseek/deepseek-chat  # Specify model
cip --dir /path/to/project       # Working directory
cip --api-key your-key           # Custom API key

Project Isolation

Common Issues

Command not found: cip

Add Python Scripts to PATH. Windows: %LOCALAPPDATA%\Programs\Python\Python312\Scripts

Use cip not cipher

Windows has cipher.exe (file encryption). Use cip.

Connection refused

For Ollama: ollama serve. For cloud providers, check your internet connection.
