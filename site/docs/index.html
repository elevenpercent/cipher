Overview

Cipher is an autonomous coding agent that runs in your terminal. You describe what to build, and Cipher plans the work, writes code, runs commands, and iterates until the task is done.

It works with 14+ AI providers out of the box. The free Cipher Proxy gives you Llama 3.3 70B and Gemini 2.0 Flash with zero configuration — no API key needed.

The interface has a sidebar for sessions and agent modes, a live streaming chat area, and a hint bar at the bottom. Use slash commands or Ctrl+P to access all actions.

Install

Python 3.10+ required. Choose your method:

pip (recommended):

pip install --upgrade git+https://github.com/elevenpercent/cipher.git@master

npm:

npm install -g https://github.com/elevenpercent/cipher.git

Windows (one-liner):

powershell -c "irm https://cipher.elevenpct.com/install.ps1 | iex"

macOS / Linux:

curl -fsSL https://cipher.elevenpct.com/install.sh | bash

Setup

Run cip in any project directory. On first launch, the setup screen appears automatically. If you have a GPU with sufficient VRAM, local Ollama models will be detected and shown. Otherwise, cloud providers are listed.

Proxy (no key)

Cipher Proxy is the default provider — free, no API key. It routes through proxy-blue-kappa.vercel.app and includes:

  • Llama 3.3 70B via Groq (~0.70s response)
  • Llama 3.1 8B via Groq (~0.23s response)
  • Gemini 2.0 Flash via Google (free tier, 60 RPM)

Bring your own key

Select any provider from the setup screen and enter your API key. Supported providers include OpenAI, Anthropic, Groq, Gemini, DeepSeek, OpenRouter, Together, Mistral, xAI, Fireworks, Perplexity, Cohere, Ollama, and LM Studio.

First run

Navigate to a project directory and run:

cd your-project
cip

The setup screen appears. Pick a provider (Cipher Proxy is default) and a model. That's it — you're ready to code.

Settings

Press Ctrl+S to open the settings modal. It has three sections:

Provider & Model

Select your AI provider and model from the dropdowns. Switching providers updates the available model list immediately.

Display

Toggle what you see in the chat:

  • Show plan — display Cipher's reasoning steps
  • Show code blocks — display code as it's written
  • Show diff — display file diffs
  • Show tool results — display command output
  • Compact mode — reduce vertical spacing
  • Auto-confirm — skip tool execution confirmation

Actions

  • Clear Chat — wipe current conversation
  • New Session — start a fresh session
  • Sessions — browse and resume previous sessions
  • Quit — exit Cipher

Proxy

The Cipher Proxy is a Vercel serverless function at proxy-blue-kappa.vercel.app. It forwards requests to Groq (Llama 3.3 70B, Llama 3.1 8B) and Google Gemini (2.0 Flash).

If the upstream provider returns an error, it's forwarded as-is. Gemini's free tier is limited to 60 requests per minute — if you see HTTP 429, wait a moment and try again.
