Features

Everything you need to build autonomously, right from your terminal.

🖥️

Terminal Native

Runs anywhere you have a terminal. No IDE plugin, no browser extension. Just `cip` and go.

🔧

Autonomous Agent

Cipher plans, writes, and executes code autonomously. It reads your files, runs commands, and iterates.

🔒

Local First

Run models locally with Ollama. Your code never leaves your machine. Complete privacy.

🔑

Bring Your Own Key

Use any cloud provider — Groq, DeepSeek, Gemini, OpenAI, Anthropic, and more. Free tiers available.

⚡

Smart TUI

Animated thinking spinner, slash autocomplete, session history, provider panel, and settings — all keyboard shortcuts.

💾

Session History

Every conversation is saved automatically. Resume where you left off, review past sessions, switch contexts instantly.
