Your terminal. An AI agent.

Cipher is an autonomous coding agent that runs entirely in your terminal. It plans, writes code, runs commands, and iterates — with any AI provider. No GPU required.

Install Features
$ pip install cipher-agent
→ One command. Any OS.
$ cd your-project && cip
→ Pick a provider. Start coding.
✓ Cipher ready.

Features

What Cipher does out of the box.

No GPU needed

Use free cloud APIs (Groq, Gemini) or run models on CPU with Ollama. No expensive hardware required.

Terminal-native

No IDE, no browser. Works anywhere you have a terminal. Launch with cip in any project directory.

Autonomous agent

Reads files, writes code, runs commands, and iterates. Plans multi-step tasks, verifies results, and adapts.

14 providers

Ollama, Groq, DeepSeek, Gemini, OpenAI, Anthropic, OpenRouter, and more. Switch with one command.

Smart TUI

Keyboard-driven interface with autocomplete, provider panel (Ctrl+P), settings (Ctrl+S), and session history.

Permission rules

Define auto-allow/deny patterns for actions. Always allow read *.py, always deny run rm*.

Custom tools

Define your own tools in config — map any shell command to a named tag. The agent discovers and uses them.

Session persistence

Every conversation auto-saves. Resume any session later. Search past conversations with /sessions.

No GPU? No problem.

Cipher works great without a graphics card. Here's how:

Free cloud APIs

Providers like Groq give you free inference — no GPU needed. Responses arrive in seconds over the internet. Run cip, pick a free provider, and go.

Small local models (CPU)

Ollama runs models like qwen3:1.5b or phi4:latest entirely on CPU. Slower but fully private. No internet, no GPU, no cost.

Mini models for speed

Models under 4B parameters (like llama3.2:3b, gemma3:4b) run comfortably on CPU with reasonable speed. Good for planning and code review on modest hardware.

Free tiers with keys

Gemini offers 60 requests/minute free. Groq and Fireworks AI offer free inference tiers. Just sign up, paste the key, and code.

Install

Python 3.10+ required. Choose your OS:

Windows (PowerShell)

One command — downloads and installs automatically:

powershell -c "irm https://cipher.elevenpct.com/install.ps1 | iex"

Or manually: pip install git+https://github.com/elevenpercent/cipher.git

macOS / Linux

One command — downloads and installs automatically:

curl -fsSL https://cipher.elevenpct.com/install.sh | bash

Or manually: pip install git+https://github.com/elevenpercent/cipher.git
