Install

Python 3.10+ required. Choose your path.

pip (recommended)

Works everywhere — Windows, macOS, Linux.

pip install git+https://github.com/elevenpercent/cipher.git@master

npm

If you're in the Node.js ecosystem.

npm install https://github.com/elevenpercent/cipher.git

Windows (PowerShell)

One-liner — downloads and installs.

powershell -c "irm https://cipher.elevenpct.com/install.ps1 | iex"

macOS / Linux

One-liner — curl pipe to bash.

curl -fsSL https://cipher.elevenpct.com/install.sh | bash

After installing, run cip in your project directory. The setup screen will appear on first launch.
